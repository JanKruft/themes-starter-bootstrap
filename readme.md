## them.es<sup>+</sup>

**them.es Starter** is a WordPress Starter Theme built with Bootstrap. Please note that the Source files is only recommended for WordPress Developers who are searching for a simple, solid, proved and tested **Bootstrap Starter Theme** to build upon. **_Don't_ expect a ready-to-use WordPress Theme!**

If you want to see it in action, check out [http://them.es/starter](http://them.es/starter)


## What's included?
* WordPress Theme
* Bootstrap Framework
* Customization API
* 2 Menus
* Custom WordPress Nav Walker class
* LESS Source files
* Demo Content


## LESS
**`main.less`** needs to be compiled as **`/css/main.css`**


## Free Add-ons Plugin
We are also running a Sister project which brings useful WordPress Add-ons to Themes built with Bootstrap. For more information, have a look at [https://wordpress.org/plugins/themes-plus](https://wordpress.org/plugins/themes-plus) or [https://github.com/them-es/themes-plus](https://github.com/them-es/themes-plus)
